Getting Started

Postico is a database client. Postico can connect to a local PostgreSQL server running on your Mac, or to remote servers running on a different computer. If you want to install a local PostgreSQL server on your Mac, I recommend Postgres.app.

We designed Postico to be easy to use for newcomers and specialists alike. If you run into any problems at all, or have suggestions for improvements, please contact us via email: postico@eggerapps.at

There is also a public issue tracker on Github. You can use it for reporting bugs, for feature requests, or if you have any questions concerning Postico's usage.
